terraform_distribution_publisher

terraform_distribution_publisher(name, deps, env, prepublish_builds, prepublish_tests)

Attributes

name

Name; Required

A unique name for this rule.
deps

List of labels; Required
env

Dictionary mapping strings to string; Optional; Default is {}

Environment variables set when publishing (useful in conjunction with '--workspace_status_command' script)
prepublish_builds

List of strings; Optional; Default is ['//...']

Ensure these target patterns build prior to publishing (eg make sure '//...' builds)
prepublish_tests

List of strings; Optional; Default is ['//...']

Ensure these tests pass prior to publishing (eg '//...', plus explicitly enumerating select tests tagged as 'manual')
